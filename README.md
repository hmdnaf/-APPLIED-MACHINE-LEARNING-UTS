Proyek ini merupakan implementasi studi kasus Data Science di bidang pendidikan/pelatihan menggunakan metodologi CRISP-DM (Cross Industry Standard Process for Data Mining).

Tujuan utama proyek adalah memprediksi probabilitas kelulusan (Lulus/Gagal) peserta pelatihan berdasarkan data ujian (UTS, UAS) dan demografi. Model ini dapat digunakan untuk identifikasi dini peserta berisiko sehingga intervensi dapat dilakukan tepat waktu.
